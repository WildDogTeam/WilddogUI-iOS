WilddogArrayDelegate Protocol Reference

Conforms to NSObject
Declared in WilddogArrayDelegate.h

Overview

A protocol to allow instances of WilddogArray to raise events through a delegate. Raises all Wilddog events except WEventTypeValue.

– childAdded:atIndex:

Delegate method which is called whenever an object is added to a WilddogArray. On a WilddogArray synchronized to a Wilddog reference, this corresponds to an WEventTypeChildAdded event being raised.

- (void)childAdded:(id)object atIndex:(NSUInteger)index

Parameters

object

The object added to the WilddogArray
