WilddogArray Class Reference

Inherits from NSObject
Declared in WilddogArray.h

Overview

WilddogArray provides an array structure that is synchronized with a Wilddog reference or query. It is useful for building custom data structures or sources, and provides the base for WilddogDataSource.
